# CRM Data Analysis Project

This project analyzes CRM data from an online programming school.

## Project goal
Identify inefficiencies in customer acquisition and deal processing and analyze factors affecting conversion to payment.

## Tasks
- data cleaning
- exploratory data analysis (EDA)
- anomaly detection
- hypothesis testing
- forecasting (ARIMA)

## Tools
Python, Pandas, SQL, statistical tests

## Key insights
- many deals are lost due to weak lead processing
- paid campaigns convert better than organic campaigns
- increasing lead volume does not necessarily increase payments
